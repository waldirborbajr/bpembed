# bpembed

BPEmbed it is a library to read a file and return the content


## Download

```go get -u github.com/waldirborbajr/bpembed```

## Using
`
import "github.com/waldirborbajr/bpembed"
`

`
fmt.Println(bpembed.BPEmbed("file.txt"))
`
